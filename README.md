# Optimize LLM Costs Token Counting

This example demonstrates how to optimize Large Language Model (LLM) costs by managing token usage. It illustrates the impact of prompt engineering (making prompts concise) and controlling output length on the total token count and hypothetical API costs, using the `tiktoken` library for accurate token calculation. The goal is to show practical strategies for reducing LLM expenses without sacrificing the quality of the results.

## Language

`python`

## How to Run

1. Ensure Python 3 is installed.
2. Install the required library: `pip install tiktoken`
3. Run the script: `python main.py`
